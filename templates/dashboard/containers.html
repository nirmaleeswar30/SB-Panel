Tips & Information
Container Types

Different container templates are available for specific use cases:

  • NGINX: For modern web applications
  • Apache: For PHP and traditional websites
  • Mixed: For multiple services on one container
Resource Management

Containers have resource limitations to ensure fair usage:

  • CPU Cores: Number of virtual CPU cores
  • Memory: RAM allocated to the container
  • Disk: Storage space available
Important Notes

Keep in mind these considerations:

  • Stopping a container will interrupt all services
  • Container deletion removes all associated data
  • Resource limits are enforced per user account
